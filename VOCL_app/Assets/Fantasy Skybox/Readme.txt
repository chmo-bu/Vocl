------------------------------------
Fantasy Skybox 1.6.0
------------------------------------

	Fantasy Skybox is ready-to-use skyboxes for Unity. It contains 178 cubemap and 160 panoramic skyboxes.

	Try Fantasy Skybox FREE for free version.

	Features:

		• 178 cubemap skyboxes.
		• 160 panoramic skyboxes.
		• Sun/sunless and moon/moonless.
		• Day, night, rainy snowy, sunrise, sunset.

	Instructions:

		1. Open Unity 2021.1.0f1 or higher then import the asset.
		2. Open your scene or the prepared demo scene.
		3. Select the Camera to make sure that the Clear Flags was set to the Skybox mode.
		4. On top menu, select Window -> Rendering -> Lighting.
		5. The Lighting dialog appears, select Environment tab.
		6. Change the Skybox Material field in the Environment tab by selecting a material in Panoramics folder for panoramic skybox or in Cubemaps folder for cubemap skybox.
